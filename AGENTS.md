# Repository Router

- 回答と文書本文は原則として日本語で書く。
- このリポジトリの persistent knowledge root は `knowledge/`。
- wiki の `bootstrap`, `ingest`, `query`, `draft-review`, `canonicalize`, `lint` を行う場合は、先に `knowledge/AGENTS.md` を読む。
- durable な設計文書、ADR、roadmap、implementation plan は `knowledge/AGENTS.md` の routing に従って `knowledge/` 配下へ保存する。
- repo root には wiki-wide な運用ルールを複写しない。
